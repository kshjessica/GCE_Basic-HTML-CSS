## 🧐 Before We Get Started
### 🔨 Make a Repository
*You need a Github account to follow this*  

1. Create a new public repository     
   Please make sure to select **Public** and to initialize the repository with a **README**
   <p align="center"><img src="https://user-images.githubusercontent.com/59790777/168459133-25a30b66-ede6-445e-82dd-84133ca61978.png" width="300" alt="Change repository visibility" /></p>
2. Navigate to your new repository

### ⚙️ Open Github.dev
1. Swap `.com` with `.dev` in the URL
   <p align="center"><img src="https://user-images.githubusercontent.com/59790777/168459239-018e7210-9a0d-4bab-b23b-0e47d8005385.png" width="300" alt="Github.dev"/></p>
3. Install CodeSwing
   <p align="center"><img src="https://user-images.githubusercontent.com/59790777/168459278-92f4a2dc-4e93-4cb7-9c51-fe41d0e24814.png" width="300" alt="CodeSwing"/></p>
4. Set up the base code through Github [Gist](https://gist.github.com/kshjessica/74e42fd2a589fa6aa844219f52cf0f32)   
   You need to make two files: `index.html`, `style.css` 
   <p align="center"><img src="https://user-images.githubusercontent.com/59790777/168459559-b4b5d285-20b7-4155-b6fa-d4e2d770f939.png" width="300" alt="files"/></p>

### ✔️ Create a website
1. Commit your changes
2. Set up Github Pages  
   <p align="center"><img src="https://user-images.githubusercontent.com/59790777/168459622-c9244daa-5829-4629-bc85-d9d26c170ee2.png" width="600" alt="files"/></p>
   (1) Navigate to your repository on GitHub.com, and select Settings, next to the gear icon.<br>
   (2) Select Pages in the left-hand menu.  <br>
   (3) Under Source select the main branch of your repository to be the source for your new website. Then, select Save.  <br>
   (4) After you select your source, you'll be notified of the address of your new website. It should look something like your-username.github.io/resume/.  
3. Navigate to the page, and check out your live resume website!
